BIOGRAPHY

プロフィール

名前 紫苑（Sion908）
役割 えぬじー / エンジニア
拠点 長崎県長崎市・群馬県（フルリモートワーク）
専門領域 Python / Litestar、AWS、メタバース制作、PWA開発、地域向けデジタル実装
キーワード Python / Remote Dev / 3DCG / 地域DX

長崎大学大学院総合工学研究科で電気電子工学を専攻。独学のエンジニアリングと、地域活動から得た現場感を掛け合わせ、 えぬじーでは「小さく作って育てる」開発を担当しています。技術選定から設計、実装、運用までを一貫して担い、 使う人の負担を増やさずに価値を届けることを最優先に考えています。

地域団体と連携したアプリの開発や、メタバース構築、オンライン花火大会、LINE連携クーポンシステムなど、 アイデアをプロダクトに落とし込む段階で伴走しながら実装するのが得意です。

VISION & VALUES

大切にしていること

VISION

地方からでも、楽しみながら続く仕組みをつくる

地方からでも、楽しみながら関わり続けられる仕組みをつくり、場所や規模に左右されず自然に続いていく状態を価値あるものとして捉えている。

METHOD

楽しさを軸にしつつ、続く形へ落とし込む

興味や楽しさを出発点にしつつ、勢いだけで終わらないよう動かしながら整え、無理なく続く形へ落とし込んでいく。

FOCUS

感情が向くほうへ進み、現場を忘れない

最終的には感情が向く方向を信じながらも、机上で完結させず現場や状況に立ち戻り、今の条件に合っているかを見て進んでいる。

WORKS

主なプロジェクト

01

まめここアプリ（ライフログPWA）

Litestar + AWS Lambda + CloudFront で構築した家族向けライフログアプリ。 Service Worker と Manifest によるホーム画面追加・オフライン対応を備え、感情を端末内にそっと残せる体験をデザイン。 サイトを見る

02

居留地オンライン花火大会

コロナ禍で失われた地域行事をオンラインで再現。CAD・Blender・UE4で街並みと花火を表現し、AviUtlで仕上げた映像を配信。 参加者が描いた絵やメッセージを花火に変換する仕掛けで、地域と医療従事者にエールを届けました。 動画を見る

03

地元のまちづくり

地元群馬の片田舎にて、地域活性化プロジェクトに取り組んでいます。 とりあえずは、地元を出た友人たちが少しでも戻る機会を作ること。 徐々に、村が活気を取り戻しつつ、正しい終わりに至る道筋を一緒に考えたいと思っています。
